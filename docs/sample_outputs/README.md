# Sample Outputs

This folder contains a compact demonstration bundle generated from the built-in fund analysis workflow.

The files are intended for product review, GitHub preview, and local smoke testing:

- `analysis_reports.pdf`: reader-friendly report bundle.
- `analysis_reports.docx`: editable Word report bundle.
- `analysis_data.xlsx`: workbook with ranking, metrics, model, and portfolio tables.
- `risk_return_balanced.png`: risk-return scatter chart.
- `top_scores_balanced.png`: top fund multi-factor score chart.
- `portfolio_optimized_weights.png`: constrained portfolio weight chart.
- `lime_local_weight_bars.png`: LIME local factor explanation chart.

These files are demo artifacts only. They are based on historical data and are not personalized investment advice, a return guarantee, or a buy/sell signal.
